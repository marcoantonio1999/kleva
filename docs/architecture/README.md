# Architecture Diagrams

This folder contains Mermaid diagrams for the voice platform architecture.

- `01-current-architecture.mmd`: Baseline production architecture currently implemented.
- `02-target-architecture-scale.mmd`: Scaled production architecture for high concurrency and operational reliability.

## Intent

1. Document the baseline system flow and current service boundaries.
2. Document the scaled architecture pattern for growth in traffic and tenants.
3. Keep reliability, observability, security, and delivery automation as first-class concerns.
